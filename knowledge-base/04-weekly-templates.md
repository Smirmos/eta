# Planning a Week

> Source: Friel 5th ed., Chapter 8, pp. 195–239 (PDF page numbers)
> Extracted: 2026-05-05

## Summary

Chapter 8 specifies how to convert the seasonal ATP from Chapter 7 into
weekly training routines. Load-bearing content for plan generation:
(1) the per-phase weekly templates in Tables 8.6A–F (pp. 238–239),
which assign **ability categories** (basic vs. advanced) to specific
weekdays for each season period; (2) the volume-distribution lookup
tables 8.4 (Daily Hours, pp. 228–229) and 8.5 (Daily TSS, pp. 231–232),
which split a weekly volume across Monday–Sunday; (3) the "5-2 training"
distribution rule (5 easy + 2 hard days weekly, p. 209); and (4) atomic
workout-placement rules (anchor workouts, breakthroughs, recovery
spacing, brick scheduling, long-ride/long-run order). The chapter
intentionally does **not** prescribe specific workout codes per day —
those come from Appendices B/C/D/E using the ability category named in
the cell. Preparation and Transition periods deliberately have **no**
weekly template; they are unstructured by design.

Page-citation convention: this file cites PDF page numbers (per
EXTRACTION_PROMPT rule 8). PDF p. 194 is the OceanofPDF watermark
(blank). Chapter 8 body content begins at PDF p. 195 ("PLANNING A
WEEK").

## ATP weights and abilities scheduling (precondition for weekly templates)

Before assigning a weekly template, Chapter 8 specifies what to write in
the ATP "Weights" column and which ability columns to mark per phase.
This determines what "basic" and "advanced" cells in the weekly
templates resolve to.

### Weights column by phase (pp. 198–199)

> "Start at the top of the ATP in the 'Weights' column. Record the
> abbreviation, as described in what follows, for each strength period,
> and work your way down through each week of the season." (p. 198)

| Phase | Weights code | Frequency | Notes |
|---|---|---|---|
| Preparation | **AA** ("anatomical adaptation") | 2–3 sessions/week | "You will probably need two weeks of AA." If >2 weeks remain in prep, also write **MT** ("muscular transition") for remaining weeks (2 weeks, 4–6 total sessions). If <2 weeks remain in prep, put MT into the first part of base 1. (p. 198) |
| Base 1 | **MS** ("maximum strength") | "the most challenging strength workouts of the season"; 2 days/week (per p. 211) | Write in 4 weeks of MS. If <4 weeks remain in base 1, roll remaining MS into base 2. "You should complete MS before starting the hard swim, bike, and run training of the base 3 period." (pp. 198–199) |
| Base 2 | **MS** (rolled over) or **SM** | varies | Per Table 8.6B (p. 238), Base 2 example shows SM. Rollover from base 1 may be present. |
| Base 2, 3, Build 1, Build 2, Peak | **SM** ("strength maintenance") | once per week | "you are lifting weights only once per week. So the strength workouts are now not only less frequent but also much easier, allowing you to train hard in the pool and on the road." (p. 199) |
| Race | (blank) — or **SM** for second of consecutive race weeks | n/a | "if it lasts only one week, leave the 'Weights' column blank. ... But if you have two or three races in consecutive weeks, then add SM for the second week only." (p. 199) |
| Transition | (blank) | n/a | "There is no need to lift weights during this period because the focus is on rest and recovery." (p. 199) |

> "What you must avoid is having a few weeks when both strength training
> and triathlon-specific training are at high workloads." (p. 199)

> "After the first A race ... it's generally a good idea to include a
> few sessions of MS. Of course, there may not be enough time if your A
> races are separated by only a few weeks. In that case, something has
> to give, and the logical candidate is strength training. So either
> record SM or leave the column blank." (p. 199)

[FLAG: The chapter labels MS/SM as "phases" in the same sense as
Chapter 7's Base/Build periods, but they overlap: SM continues across
Base 2 → Peak. The plan generator should treat MS/SM/AA/MT as a
**parallel** weights periodization track, not as an alternative to the
sport-specific period.]

### Ability columns by phase (pp. 200–205)

For each week's row on the ATP, mark an X in ability columns under
Swim, Bike, Run.

| Phase | Abilities marked (per week, all 3 sports) | Testing | Notes |
|---|---|---|---|
| Preparation | Aerobic Endurance, Speed Skills | last prep week: also Testing | "The only sport-specific abilities to be included now are 'Aerobic Endurance' and 'Speed Skills.'" (p. 200) |
| Base 1 | Aerobic Endurance, Speed Skills | last base 1 week: also Testing | "the emphasis is definitely on strength training... so the sport-specific workouts are quite easy. As with the prep period, place an X in the 'Aerobic Endurance' and 'Speed Skills' columns." (p. 201) |
| Base 2 | Aerobic Endurance, Speed Skills, Muscular Force, Muscular Endurance | last base 2 week: Aerobic Endurance, Speed Skills, Testing | "Mark each of these columns with an X for every week of base 2 except for muscular force sessions, which may be omitted as long as you are still doing MS sessions in the gym." (p. 201) |
| Base 3 | Aerobic Endurance, Speed Skills, Muscular Force, Muscular Endurance | last base 3 week: Aerobic Endurance, Speed Skills, Testing | "start by marking the ability columns for each sport exactly as you did in base 2." For long-course + highly competitive athletes: also mark **Aerobic Capacity** for all three sports each week except the last week of the period. (p. 202) |
| Build 1 | All abilities (Aerobic Endurance, Speed Skills, Muscular Force, Muscular Endurance, Aerobic Capacity) | last build 1 week: Aerobic Endurance, Speed Skills, Testing | "Put an X under every ability column for each of the three sports for both build 1 and build 2 for all but the last weeks of each period." (p. 202) |
| Build 2 | All abilities (same as Build 1) | last build 2 week: Aerobic Endurance, Speed Skills, Testing | (same rule as Build 1, p. 202) |
| Peak | **Long-course (half-IM, IM): Muscular Endurance only.** Sprint/Olympic: Muscular Endurance + Anaerobic Endurance. | "There is no testing in the peak period. The race will be your test." | "Aerobic endurance and muscular force should be quite well developed by now. The long-course triathlete will not do aerobic capacity training at this time, but it continues for short-course racers." (p. 205) |
| Race | mark "just as you did for the peak period" | n/a | "The emphasis in your race week is on rest, with workouts that are quite brief but still focused on intensity." (p. 205) |
| Transition | (no abilities marked, leave blank) | n/a | "Do not mark any abilities in this period. Leave the spaces blank. ... any training or exercise you do should be an unplanned, spur-of-the-moment decision and should be short and easy." (p. 205) |

> "Notice on the ATP that sprint power is also listed as an ability
> under each sport, but we won't use it because it has little value for
> nondrafting races." (p. 200)

> "After the first A race of the season, you need to test aerobic
> endurance." (p. 205, callout) — operationalized as: do an AE2 workout
> (see Appendices B, C, D) and compare efficiency factor to last base 3
> best workout. If EF is ≥10% lower, return to base 3 after the
> transition until EF is <5% lower. If EF is at the same level, go
> straight into a build period. (p. 206)

### Period progression by sport (p. 207)

Sports may progress at different rates. Adjustments:

> "It may be necessary to do basic-ability training, especially for
> aerobic endurance, in one of the sports while advanced-ability
> training is being done for the other two." (p. 207)

> "If two of your sports need to return to the basic abilities, then
> it's best to return to the base 3 period for all three of them."
> (p. 207, callout)

> "The 12 weeks of base 1, 2, and 3 are usually enough to develop all
> the abilities in all three sports." (p. 207, callout)

## Weekly template structure

Friel provides **two parallel sets** of weekly templates that the plan
generator must use together.

**Set 1 — Tables 8.6A–F (pp. 238–239):** the canonical per-phase
templates. Cells contain ability category labels, not workout codes.
Resolve "Basic" / "Advanced" via Chapter 6 categorization (also stated
on p. 238): basic = aerobic endurance, speed skills, muscular force;
advanced = muscular endurance, aerobic capacity.

**Set 2 — Tables 8.1, 8.2, 8.3 (pp. 216, 218–219, 220–221):** the
"hard/easy" example templates that show the same weekly skeleton in
breakthrough-vs-recovery terms.

The two sets do **not** explicitly map to one another in the source.
[FLAG: Tables 8.6A–F use "Basic"/"Advanced" terminology; Tables 8.1–8.3
use "hard"/"easy"/"BT". The chapter uses both vocabularies but does not
print a unified mapping. Plan generator should treat the 8.6 series as
the ATP-aligned template and the 8.1–8.3 series as the hard/easy
overlay on top of it.]

> "Workouts in these figures are listed as basic (aerobic endurance,
> speed skills, muscular force) or advanced (muscular endurance,
> aerobic capacity) abilities. (Chapter 6 describes these workout
> categories, and Appendixes B, C, D, and E provide the specific
> workouts within each category.) Making a choice for which specific
> type of workout to do on a given day depends on your limiters,
> activity availability (such as gym, pool, or masters swim days), and
> the race distance for which you are training." (p. 238)

> "Note that the preparation and transition periods are not included in
> these figures because they are highly variable; they depend on how
> you feel on any given day. They are also unstructured." (p. 238)

### Peak-period taper rule (p. 217)

> "Starting two to three weeks before your A-priority race, do a
> race-intensity workout every third day that simulates the conditions
> of the race. Then make these workouts gradually shorter as you
> progress through the peak period. With the workouts becoming shorter,
> your weekly volume is also dropping. That's good. It should drop
> rather rapidly. Something such as a 30 to 50 percent drop each week
> is about right." (p. 217)

> "The intensity for these race-like workouts should be similar to what
> you're expecting in the race, but the intensity must reach at least
> zone 4 for heart rate, pace, or power — 'hard.'" (p. 217)

> "The two recovery days between these race simulations are key to
> reducing fatigue while elevating form and maintaining aerobic
> endurance. They should be low-intensity, low-duration workouts that
> also become shorter as the peak period progresses." (p. 217)

### Race-week rule (p. 218)

> "In race week, do three or four workouts in which you include several
> 90-second intervals with 3-minute recoveries following a warm-up. Cool
> down afterward. For short-course races, the intensity is 'hard' to
> 'really hard' perceived exertion, or zone 4 or 5 for power or pace
> (see Chapter 4 for details on intensity). For long-course races, the
> intensity is zone 3 power or pace, or a perceived exertion of
> 'somewhat hard.' Do not use heart rate to gauge intensity for these
> intervals; they are too brief. Your heart would not have enough time
> to respond and achieve the targeted zone in only 90 seconds." (p. 218)

> "Over the course of the week, you need to decrease the number of
> intervals, which means that the individual workout times will also
> gradually decrease." (p. 218)

### Taper-length adjustments by sub-discipline (p. 217)

> "running, which is more orthopedically stressful, typically requires
> a longer taper than does cycling, and the cycling taper is usually
> longer than that for swimming." (p. 217)

> "Consider other elements... such as the length of the race
> (long-course races mean long tapers), how fit you are (high fitness
> means long tapers), how easily injured you are (injury-prone athletes
> should taper longer), and how old you are (older athletes often need
> longer tapers)." (p. 217)

## Workout placement rules

Atomic, source-cited rules ready for plan-generator implementation.
Each rule: trigger condition + placement constraint + page citation.

### Intensity distribution

1. **5-2 distribution (p. 209).** "The method I use to simplify this is
   called '5-2' training. This calls for doing five easy (zones 1 and
   2) weekly and two hard (zones 4 and 5) days weekly." (p. 209)

2. **Polarized 80-20 (p. 209).** "a fairly new way to do this is called
   'polarized' training, or an '80-20' distribution of low-intensity
   (zones 1 and 2) and high-intensity (zones 4 and 5) workouts."
   (p. 209)

3. **Hard-day workout count (p. 209).** "High-performance triathletes
   usually do two workouts daily. This means that on hard days one or
   two zone 4 and 5 workouts are done. ... Otherwise, on a hard day,
   one or both workouts may be hard depending on the athlete's capacity
   for workload and speed of recovery. If in doubt, do only one hard
   workout on these days." (p. 209)

### Anchor workouts

4. **Anchors are placed first (p. 209).** "'Anchors' are workouts that
   are tied to specific days and times. Write these into your weekly
   plan first, and build all the other sessions around them." (p. 209)

5. **Anchor stress awareness (p. 209).** "Pay attention to the common
   stress levels of these sessions, especially those that are group
   workouts. ... Once you have planned the entire week, you may see
   there are days when you need to move to a slower lane, 'sit in' on
   the ride, or run with a slower group." (p. 209)

### Breakthrough workouts

6. **Breakthrough = stressful enough to gain fitness (p. 210).** "These
   are the sessions that are stressful enough to produce greater
   fitness — you *break through* to a new level of fitness." (p. 210)

7. **Recovery before/after breakthrough (p. 210).** "The primary
   concerns here are arriving at them rested and ready, and recovering
   for a day or so afterward." (p. 210)

8. **Base-period breakthroughs target basic abilities (p. 211).** "it's
   common in the base period to have breakthrough workouts for the
   basic abilities, such as aerobic endurance and muscular force."
   (p. 211)

9. **Build-period breakthroughs target advanced abilities (p. 211).**
   "The build period is usually focused on the advanced abilities —
   muscular endurance and aerobic capacity." (p. 211)

10. **MS-week BT pairing (p. 211).** "in the early part of the base
    period — base 1 and perhaps early base 2 — you may be lifting 2 days
    each week. On your weekly routine, separate them by at least 2 days
    so that they are scheduled for Monday and Thursday, Tuesday and
    Friday, or some similar pairing that works for your lifestyle.
    Treat these as your breakthrough (hard) workouts, meaning that a
    day of recovery is needed afterward." (p. 211)

11. **Post-early-base SM frequency (p. 211).** "After the early base
    period, you will lift only once each week until the next race, and
    you should reduce the gym workouts to a moderate intensity so that
    they are no longer breakthroughs." (p. 211)

12. **Experienced-athlete BT cap (p. 212).** "experienced athletes can
    typically schedule up to four breakthrough workouts in a week on
    their two hard days. Those less experienced should do fewer,
    perhaps only one on each hard day." (p. 212)

13. **Junior/senior BT recovery (p. 212).** "both junior and senior
    athletes need longer recovery breaks from training before
    breakthrough sessions." (p. 212)

### Easy / recovery / speed skills

14. **Easy days are critical (p. 212).** "Recovery, aerobic endurance,
    and speed skills are just as critical to your fitness gains and
    your success on race day as the hard breakthrough workouts. ... Do
    not overlook the value of the easy days." (p. 212)

15. **Most easy days target aerobic endurance (p. 215).** "Nonbreak­
    through, easy workouts are intended for recovery, skill improvement,
    or aerobic endurance. In fact, most of your easy training days
    should be focused on aerobic endurance." (p. 215)

### Two-a-days and limiter ordering

16. **Two-a-days for advanced athletes (p. 212).** "It's common for
    experienced age-group triathletes to do two workouts a day. Most
    pros frequently do three-a-days. ... if you are a novice and your
    goal is to cross the finish line of a race, then multiple daily
    workouts are likely not necessary." (p. 212)

17. **Same-sport two-a-day separation (p. 212).** "A single workout each
    day, with a weekly day off from all training, will give you six
    sessions weekly with two sessions in each sport. The two
    sport-specific workouts should be separated by two days, such as
    run on Monday and Thursday, swim on Tuesday and Friday, and bike on
    Wednesday and Saturday, with Sunday a day off." (p. 212–213)

18. **Run-after-bike injury risk (p. 214).** "Running on tired legs as
    the second workout of a two-a-day, especially after cycling or
    lifting weights, is very risky for some. If this is a concern of
    yours, it's better to do the run earlier in the day when your legs
    are likely to be fresher, then ride or go to the gym later."
    (p. 214)

19. **Limiter sport scheduled first (p. 226).** "Doing two-a-day
    sessions increases the importance of your limiting sport when it
    comes to scheduling. If you need to improve in one sport far more
    than the other one planned for that same day, you should take into
    consideration the order in which they will be done. It's
    undoubtedly better to tackle the workout for your limiting sport
    first, especially if your energy and motivation after the first
    workout are diminished... You don't want to go into breakthroughs
    tired. Hence, the breakthrough session for the limiting sport is
    best done before the other session." (p. 226)

20. **First workout = active recovery → order doesn't matter (p. 229).**
    "If the first session is only for active recovery — meaning short
    and with low intensity — then the order of scheduling is of little
    concern." (p. 229)

21. **Both BT, late-day person → BT goes second (p. 229).** "If you are
    not a morning person, though, and tend to have higher-quality
    workouts later in the day, then you should schedule the
    breakthrough workout for your limiting sport later in the day. In
    this situation, it's critical that you make the first workout easy
    enough that it doesn't hurt the breakthrough workout for your
    limiter." (p. 229)

### Sports distribution

22. **Minimum 3 workouts per sport (p. 214).** "For the highly
    experienced triathlete, I'd suggest doing at least three workouts
    in each sport weekly, with a day or more of separation between
    those in a single sport." (p. 214)

23. **Extras go to bike or limiter (p. 214).** "If you can manage more
    than nine sessions in a week, then the 'extra' ones should be on
    the bike or in your limiting sport." (p. 214)

24. **Why bike (p. 214).** "About half of a triathlon is spent cycling,
    so it's the sport that contributes the most to your race
    performance. Cycling is also a low-stress workout that builds
    aerobic endurance that will benefit both swimming and running."
    (p. 214)

### Long ride / long run ordering (load-bearing for plan generation)

25. **Anti-pattern: long bike Saturday → long run Sunday is wrong
    (p. 232).** "The thinking is usually along this line: *I will be
    tired when I start the run in a race; therefore, I need to do my
    long run on Sunday when I'm tired from the long ride on Saturday.*
    But that's wrongheaded thinking. The fatigue you experience while
    running the day after a long ride is not the same as the fatigue
    you feel as soon as you exit T2 in a race." (p. 232)

26. **Long run on Sunday after long Saturday ride causes injury risk
    (p. 232).** "a long run on Sunday following a Saturday long ride
    increases your risk for an injury while lowering the quality of
    your run. Running injuries are common among triathletes. And
    running on tired legs is a common cause of injury. If running is a
    limiter, then doing that workout when leg-weary on Sunday will have
    little benefit." (p. 232)

27. **Best solution: split long ride and long run across the week
    (p. 232).** "The best solution is to do one of the long workouts —
    probably the run because it's a bit shorter than a long ride — on
    Tuesday, Wednesday, or Thursday. That way, with the long ride
    scheduled for the weekend, you will probably be fresh for both of
    them." (p. 232)

28. **If both long workouts must be on the same weekend → long run
    Saturday, long ride Sunday (p. 232).** "If you must do both long
    workouts on the same weekend, I'd suggest doing the long run on
    Saturday and the long ride on Sunday." (p. 232)

29. **Alternate long bike weekend / long run weekend (p. 232).**
    "Another common solution is to do the long bike workout on one
    weekend and the long run the following weekend." (p. 232)

### Brick workouts

30. **Bike-run bricks belong in build/peak (p. 232–233).** "You can do
    the bike-run brick year-round, but it's most effective in the build
    and peak periods when workouts are becoming increasingly
    race-like." (p. 232)

31. **Short-run bricks > long-run bricks (p. 233).** "It takes only 15
    to 20 minutes of running off the bike to reap the benefits of a
    brick workout. Going longer may be mentally beneficial, but there's
    little to be gained physiologically. ... A steady diet of long runs
    following long rides is a sure way to experience a breakdown of
    some sort, such as injury, illness, or low motivation. Over the
    course of the build period, frequent short-run bricks are more
    valuable than a few long-duration runs." (p. 233)

32. **T1 transition rehearsal (p. 233).** "You should rehearse your T1
    transition several times during the peak and race periods."
    (p. 233)

33. **T2 rehearsal during every late-build ride (p. 233).** "T2
    rehearsal starts even before you get off the bike. Near the end of
    every ride in the build period, take your feet out of your shoes
    while staying clipped in. Get used to pedaling with your feet on
    top of your shoes instead of in them. Again, that's every ride."
    (p. 233)

34. **Brick = separate workouts for planning purposes (p. 224).** "For
    the purposes of weekly planning, the multiple sports in a brick are
    considered separate workouts even though they are done as a single
    nonstop session." (p. 224)

### Aerobic capacity workouts

35. **AC workout definition (p. 234).** "An aerobic capacity workout is
    one involving short intervals (usually less than four minutes for
    each interval) done at a very high intensity — zones 5a and 5b —
    with short recoveries (again, four minutes or less)." (p. 234)

36. **Long-course AC: do in base 3, not build (pp. 234–235).** "aerobic
    capacity workouts are quite dissimilar to what can be expected in
    long-course triathlons — they aren't race-like — so such workouts
    are best done before the build periods starts. Long-course
    triathletes typically devote most of the high-intensity training
    time in the build period to muscular endurance workouts. By doing
    aerobic capacity workouts in base 3, they can lift their aerobic
    capacities to a high level before starting the build period, and
    then maintain it with less frequent anaerobic endurance intervals
    in the last few weeks before their race." (p. 235)

37. **Short-course AC: build period only (p. 234).** "short-course
    triathletes don't need to start [aerobic capacity training] earlier
    than the build period because their aerobic capacity workouts are
    similar to what they can expect in a race." (p. 234–235)

### Missed workouts (decision rules)

38. **≤3 days missed → no makeup (p. 236).** "**Missing three days or
    fewer.** Return to training as if nothing happened. Don't try to
    make up the missed workouts. Cramming more workouts into a few days
    creates the potential for a breakdown and more lost training time
    if it rarely happens. It's not a big deal to miss a handful of
    workouts." (p. 236)

39. **4–6 days missed (illness) → treat as 7+ days (p. 236).** "If the
    lost time was due to illness, ... you will need to treat it as more
    than seven days missed, even though you're starting back into
    training again." (p. 236)

40. **4–6 days missed (non-illness) → treat as rest-and-recovery; reduce
    period (pp. 236–237).** "consider the lost training time as a
    rest-and-recovery period. ... reduce the length of the current
    period by one week. If you still aren't synchronized, do the same
    for the following period. The second option is to reduce the peak
    period from two weeks to one week." (pp. 236–237)

41. **1 week missed → make up 2–3 key workouts (p. 237).** "you will
    need to step back and make up probably two or three key workouts
    that were missed. Decide which were the most important ones given
    your limiters and reschedule them." (p. 237)

42. **Missing 1–2 weeks in build → restart at base 3 (p. 237).** "If
    you missed this time because of illness and you were in the build
    period, start back into training with the base 3 training block."
    (p. 237)

43. **Missing 1–2 weeks in base → back up one period (p. 237).** "If
    you were in the base period, go back to the previous base period or
    even the preparation period if you were in base 1." (p. 237)

44. **Hold backup period 3–4 weeks (p. 237).** "Stay with that period
    for three to four weeks or until your workouts indicate that you
    are back to where you left off before the lost training. You will
    know you've reached that level because your heart rate and
    perceived exertion will again match pace and power, as they did
    before you got sick." (p. 237)

45. **Repeat last hard week before resuming (p. 237).** "When your
    training metrics and vigor return to normal, repeat the last week
    of hard training you did before the interruption. If that week goes
    well, then begin moving forward with your original training plan
    but start from a later time in the season." (p. 237)

46. **Long break: omit some training (p. 237).** "At some point, you
    will need to leave out one to three weeks (or even more) of planned
    training. That could mean omitting build 2 and perhaps the first
    week of the peak period. Make sure you complete the full base
    period, however." (p. 237)

47. **>2 weeks missed in build → return to base 3 (p. 237).** "If you
    were in the build period when this training pause happened, return
    to base 3 and start over again from there." (p. 237)

48. **>2 weeks missed in base → back up one period (p. 237).** "If you
    already were in the base period, back up one period from where you
    left off and start training again from that point." (p. 237)

49. **Omission priority: peak wk 1, build 2, build 1; preserve base
    (p. 238).** "The priority for omissions, in order, is the first
    week of peak, build 2, and build 1. Again, complete all three base
    periods." (p. 238)

50. **Last week of build 2 / peak: continue as if nothing happened
    (p. 238).** "If any of your training time was lost in the last week
    of build 2 or the peak period, continue with your training as if
    nothing had happened. But ... if the lost time was due to illness,
    be conservative with both duration and intensity as you start back
    by opting to train with short- to medium-distance workouts and
    intensity primarily in zones 1 and 2 until you are back to feeling
    normal." (p. 238)

## Daily training structure (two-a-days, AM/PM, recovery between sessions)

### Two-a-day rationale

> "The obvious advantage of doing two-a-day workouts is that you can fit
> in more training in each sport weekly than if you do only one each
> day. ... With only one session each day, you can manage only two or
> three workouts per sport in a week." (p. 213)

### Two-a-day risks

> "Doing two or three workouts in a single day means increased fatigue.
> Some athletes seem to handle that quite well and bounce back quickly.
> Others, especially novices, juniors, and seniors, recover more
> slowly. So several stressful, back-to-back, double-workout days can
> lower the quality of your training and therefore your fitness and
> performance." (p. 213)

> "two-a-day workouts raise the possibility of injury, which is
> especially common in running." (pp. 213–214)

### Bricks counted as separate workouts

See rule 34 above (p. 224).

### AM/PM ordering

See rules 19, 20, 21 above (pp. 226, 229).

### Order of sports on a single day (default)

> "Runs are typically shorter workouts than bike rides — or even swims
> when you include travel time to the pool — so your available time may
> make the decision for you." (p. 225)

> "On some days, your anchor workouts will decide this for you. For
> example, if you swim with a masters group that meets early in the
> morning on certain days, then the order of workouts that day is
> settled." (p. 225)

## Weekly volume distribution

### Volume drivers by experience level

> "for the novice athlete, the frequency of workouts should be the
> primary focus. ... For the intermediate athlete, who may be in the
> second or third year of the sport ... the most important of the
> three variables is workout duration. ... For the advanced athlete who
> has been training for three years or more, however, intensity is
> key." (pp. 220–221)

### Sport-time distribution baseline

> "For most athletes, the race time distribution is roughly 10 percent
> swimming, 50 percent biking, and 40 percent running. ... your common
> time or TSS distribution by sport gives you a good idea of how much
> volume you should devote to each sport, on average, in a standard
> training week." (pp. 222–223)

> "For example, if you expect to finish the race with 13 percent of
> your total time as swimming, then it may be a good idea to devote
> around 13 percent of your weekly training time or TSS to the swim."
> (p. 223)

> "Just don't move so much training away from your strongest sport that
> it becomes a weakness." (p. 223)

### Weekly planning procedure (p. 224)

1. Subtract anticipated weight-lifting time/TSS from the week's total
   in Table 8.4 or 8.5.
2. Divide remainder among the three sports per the race-time
   distribution (defaults: 10% swim / 50% bike / 40% run, p. 222),
   then shift up to ~10% from strongest to weakest sport.
3. Multiple sports in a brick are counted as separate workouts.
4. Place anchor workouts on their fixed days/times; subtract their
   total duration/TSS by sport from each sport's weekly volume.
5. Place breakthrough workouts on the chosen hard days; record their
   duration/TSS.
6. The remaining volume = active recovery + speed skills sessions;
   distribute over the remaining easy days.

> "You now have a detailed weekly plan. While this procedure probably
> seems quite time-consuming, after a few weeks of doing it I'm
> certain that you'll find you can do it quickly. In fact, you'll
> find, I'm sure, that there's a common pattern you follow most weeks
> of the season, which makes the process quite simple once you get on
> to it." (p. 224)

> "The only weekly routine and volume distribution we haven't covered
> so far is for the rest-and-recovery weeks, which are critical to the
> quality of your training and for avoiding overtraining. Chapter 11
> will cover how to distribute weekly volume at these times." (p. 224)

## Open questions / flags

1. **No 5/6/7 days-per-week template variants.** The CH8 instruction
   asks for variants if given. The chapter does not give them
   explicitly. The closest analogs are: the 5-2 distribution (rule 1
   above), the experienced-athlete minimum-3-per-sport rule (rule 22),
   and the single-workout-per-day option in two-a-day text (rule 17).
   Plan generator must derive 5/6-day variants by dropping
   easy/recovery sessions from Tables 8.6A–E in priority order, with
   anchor and BT cells preserved. **This is interpolation, not source
   content.**

2. **No Saturday-race variant of Table 8.6F.** Only Sunday-race shown
   (p. 239). Table 8.3 (p. 221) gives both Sat and Sun race variants
   in hard/easy terminology but not in the 8.6-series ability
   vocabulary.

3. **Table 8.2 caption/footnote contradiction.** Caption says "Two-Week
   Peak Period Routine" (p. 218); footnote says example is for "base
   and build periods" (p. 219). Resolved in favor of the caption (peak
   period). The body text on p. 218 introduces Table 8.2 in the
   context of "how a two-week peak period may be designed," confirming
   the caption.

4. **Table 8.4 / 8.5 PDF rendering overlap.** Tail rows of both lookup
   tables are partly obscured by overlapping body paragraphs in the
   source PDF. Captured rows are all clean reads up to weekly hours
   28:00 (Table 8.4) and weekly TSS 1,200 (Table 8.5). Beyond, several
   cells are marked `[UNCLEAR]`. Plan generator should re-verify tail
   rows from a higher-resolution source.

5. **Chapter content extends past the requested page range.** The user
   originally asked for pp. 194–237 per the README; the extended request
   reached p. 239. Tables 8.6A–F appear at pp. 238–239, so the extension
   captured the load-bearing per-phase weekly templates that pp. 194–237
   alone would have missed. The chapter ends mid-page on p. 239 with
   "Most important, you now have a plan for the upcoming season that
   can keep you on track for..." — text continues to a "WHEN AND HOW
   OFTEN TO TEST" or similar trailing section beyond p. 239, which is
   not part of the requested range. **Beyond p. 239 was not extracted.**

6. **MS/SM as "phases" vs. periods.** The chapter uses period vocabulary
   for both sport-specific periodization (Base 1 etc.) and weights
   periodization (AA, MT, MS, SM). They are parallel tracks, not
   alternates. Tables 8.6A and 8.6B label Monday weights as "MS phase"
   and "SM phase" respectively, which only makes sense if the reader
   already knows the weights-track schedule from pp. 198–199.

7. **Anchor-workout examples are hypothetical.** "lap swimming when it
   is available," "masters swim team sessions," "track sessions," "group
   rides" (p. 210). Plan generator must accept these as user-supplied
   constraints; the chapter gives no default schedule.

8. **5-2 vs. 80-20 relationship.** The chapter introduces both 5-2
   (p. 209) and 80-20 polarized (p. 209) but does not state whether
   they should sum identically across a week. 80-20 is a time/intensity
   distribution; 5-2 is a day-count distribution. They are compatible
   but not identical.

9. **AC for short-course in base 3?** The chapter says short-course
   athletes don't need AC training before build (p. 234). But Tables
   8.6C and 8.6D both include "Advanced" cells (which include AC) for
   all three sports. [FLAG: This is consistent only if "Advanced" in
   base 3 = muscular endurance (not aerobic capacity) for short-course
   athletes. The chapter at p. 202 confirms: "Aerobic Capacity"
   marking in base 3 is for "long course and highly competitive"
   athletes only. Plan generator must respect this conditional.]

10. **Missing-workouts rule sequence interaction.** Rules 38–50 cascade.
    A 6-day non-illness break may shorten the current period by 1 week,
    but if that current period is also peak, the rule says reduce peak
    from 2 weeks to 1 week. These rules can compound and a more general
    rewrite of the season plan may be required. Plan generator must
    apply rules in order of break length and re-validate the resulting
    ATP after each application.
